# Pax-Paarth — Marketing Website

Static single-page marketing site for **Pax-Paarth**, a premium packaging solutions company.

Designed by [Ree-Infinity Technologies](https://www.reeinfinitytech.com/).

---

## Getting Started

No build step required. Serve locally with:

```bash
python3 -m http.server 8080
```

Then open [http://localhost:8080](http://localhost:8080). Changes are live on browser refresh.

---

## Tech Stack

- **HTML/CSS/JS** — vanilla, no frameworks or package manager
- **Fonts** — Playfair Display + Inter via Google Fonts CDN
- **Icons** — Font Awesome 4.x (`css/font-awesome/`)

---

## File Structure

```
pax-paarth/
├── index.html          ← entire site (single page)
├── css/
│   ├── main.css        ← all active styles
│   └── font-awesome/   ← FA 4.x icon font
├── js/
│   └── main.js         ← all active JS (vanilla, no jQuery)
└── images/
    ├── logo.png
    ├── hero-bg.jpg
    └── image1–6.*      ← product images
```
